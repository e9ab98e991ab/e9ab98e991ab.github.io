本题考查提高质量属性的常见手段。       <br/>
提高可用性的手段包括：命令/响应机制、心跳机制、异常处理机制、冗余机制等。       <br/>
提高性能的手段包括：引入并发、维持数据或计算的多个副本、增加可用资源、控制采样频度、限制执行时间、固定优先级调度等。     <br/>
 提高安全性的手段包括：身份认证、限制访问、检测攻击、维护完整性等。</p>
